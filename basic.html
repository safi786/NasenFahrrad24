      Berechnen Sie lhre PD-Werte

      (Pupillendistanz)

      • Ihre Kamera sollte bereit sein, bitte gegebenenfalls autorisieren

      • Halten Sie eine Karte in handelsüblicher Kreditkartengröße bereit

      Berechnen Sie lhre PD-Werte

      (Pupillendistanz)

      • Pressen Sie eine handelsübliche
        Kreditkarte o.ö., fest an Ihre Lippen.
        (Die genormte Größe dient als Referenz)
      • Schauen Sie ohne Brille gerade in IhreKamera.
      • Betätigen Sie dann den button
        „Messfoto aufnehmen“,
        um ein Selfie aufzunehmen.

      Berechnen Sie lhre PD-Werte

      (Pupillendistanz)

      • Platzieren Sie die jeweiligen Marker genau auf Ihre Pupillen und die Mitte des Nasenrückens
      • Markieren Sie mit den Ecken die genaue Breite der Kreditkarte
      • Betätigen Sie dann den button und notieren Sie sich die angezeigten Werte für links und rechts
